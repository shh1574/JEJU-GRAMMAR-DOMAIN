# JEJU-GRAMMAR-DOMAIN

제주방언 문법 연구사 정리 — 형식 × 기능 인터랙티브 히트맵과 서지 정보.

## 바로 보기

### 1. 인터랙티브 히트맵 (형식 × 기능, 호버 시 논문 목록)

- **GitHub Pages (권장):** <https://shh1574.github.io/JEJU-GRAMMAR-DOMAIN/>
  - 활성화 방법: 이 레포 **Settings → Pages → Source** 에서 `Deploy from a branch` 선택, **Branch** 를 `main` / `/(root)` 로 지정하고 Save. 1–2분 후 위 URL 에서 접속 가능.
- **즉시 미리보기 (Pages 없이):** <https://htmlpreview.github.io/?https://github.com/shh1574/JEJU-GRAMMAR-DOMAIN/blob/main/index.html>

### 2. 서지 정보 표 (저자명·발행년도·제목·학회지·논문/단행본, 175편)

- **CSV (GitHub 자동 표 렌더링):** [한국어학사_제주방언 문법 연구사 정리_서지정보 260513.csv](한국어학사_제주방언%20문법%20연구사%20정리_서지정보%20260513.csv)
- **Excel 원본 다운로드:** [한국어학사_제주방언 문법 연구사 정리_서지정보 260513.xlsx](한국어학사_제주방언%20문법%20연구사%20정리_서지정보%20260513.xlsx)
